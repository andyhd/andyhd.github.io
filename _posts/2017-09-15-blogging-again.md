Here's a familiar code snippet to test things.

```python
def fizzbuzz(i):

    out = ''

    if i % 3 == 0:
        out += 'fizz'

    if i % 5 == 0:
        out += 'buzz'

    return out or str(i)


for i in range(1, 20):
    print(fizzbuzz(i))
```
